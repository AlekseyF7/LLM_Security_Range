# Инструкция по эксплуатации системы контроля и управления доступом (СКУД) "PERCo-S-20"

## Назначение системы
Система контроля и управления доступом "PERCo-S-20" предназначена для ограничения доступа в помещения ООО "ТехноЛайн", учета рабочего времени сотрудников и предотвращения несанкционированного проникновения.

## 1. Состав системы
- Контроллеры PERCo-CL201.1 (установлены на входных дверях офисов на 1, 3, 5 этажах).
- Считыватели PERCo-RP-15 (бесконтактные, стандарт EM-Marine).
- Турникеты PERCo-TTD-03A (на центральном входе).
- Программное обеспечение "PERCo-S-20 Базовая версия" (установлено на сервере srv-skud01).

## 2. Уровни доступа
В системе настроены следующие уровни доступа:

| Уровень | Наименование          | Помещения с доступом                                      |
|---------|-----------------------|-----------------------------------------------------------|
| 0       | Гость                 | Только зона ресепшн                                       |
| 1       | Сотрудник             | Офисные помещения своего этажа, кухня, переговорные       |
| 2       | Руководитель отдела   | Уровень 1 + кабинеты руководителей, комната отдыха        |
| 3       | Топ-менеджмент        | Все помещения, включая серверную (по согласованию с ИТ)   |
| 4       | ИТ-персонал           | Уровень 1 + серверная, кроссовые, помещения связи         |
| 5       | Служба безопасности   | Все помещения (включая архив и кабинет видеонаблюдения)   |

## 3. Работа с ПО "PERCo-S-20"
### 3.1. Запуск программы
Ярлык расположен на рабочем столе сервера srv-skud01. Учетная запись для входа: `operator` (пароль хранится в KeePass).

### 3.2. Добавление новой карты доступа
1. Перейти в раздел "Картотека сотрудников".
2. Нажать "Добавить".
3. Заполнить ФИО, отдел, должность, выбрать уровень доступа.
4. В поле "Код карты" поднести карту к считывателю, подключенному к серверу.
5. Нажать "Сохранить".

### 3.3. Настройка временных зон
Временные зоны определяют, в какие часы и дни недели разрешен проход. Например, для стандартного сотрудника:
- Пн-Пт: 07:00 - 22:00
- Сб: 09:00 - 18:00
- Вс: доступ запрещен

### 3.4. Формирование отчетов
- **Отчет по опозданиям**: "Отчеты" -> "Рабочее время" -> "Опоздания".
- **Отчет по присутствию**: "Отчеты" -> "Присутствие на рабочем месте".
- **Журнал событий**: "Мониторинг" -> "Журнал событий" (фиксирует все проходы, отказы в доступе, тревоги).

## 4. Действия в нештатных ситуациях
| Ситуация                               | Действия                                                              |
|----------------------------------------|-----------------------------------------------------------------------|
| Потеря карты сотрудником               | Немедленно заблокировать карту в ПО. Выдать временную карту гостя.    |
| Несанкционированное проникновение      | Нажать тревожную кнопку в ПО (или физическую под столом охраны).       |
| Отказ контроллера                      | Переключить дверь в механический режим (ключ хранится у охраны).       |
| Срабатывание пожарной сигнализации     | Система автоматически разблокирует все двери на путях эвакуации.       |
